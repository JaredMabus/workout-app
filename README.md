## Description

Single-page weight lifting app providing the ability to track progress, plan a routine, and get insight into workout patterns and metrics

## Tech Stack

- MongoDB
- Express
- React
- Node

## Usage

Run nodemon on port 3001

```sh
npm run watch
```

Build server with `tsc` to `./dist`

```sh
npm run build
```
